# AWS Serverless E-Commerce Order Processing System

## Overview

A fully serverless e-commerce order processing application built using AWS cloud services.

## Technologies Used

* Amazon S3
* Amazon API Gateway
* AWS Lambda
* Amazon DynamoDB
* Amazon SQS
* Amazon SNS
* HTML
* CSS
* JavaScript
* Python

## Architecture

Frontend Website (Amazon S3)

↓

Amazon API Gateway

↓

Lambda Function (CreateOrder)

↓

Amazon DynamoDB

↓

Amazon SQS

↓

Lambda Function (ProcessOrder)

↓

Amazon SNS

↓

Email Notification

## Features

* Static website hosting using Amazon S3
* REST API using API Gateway
* Serverless backend with AWS Lambda
* NoSQL storage using DynamoDB
* Asynchronous processing using Amazon SQS
* Email notifications using Amazon SNS
* Event-driven architecture
